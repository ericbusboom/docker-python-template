Alpine.js Interactions

This panel is toggled with Alpine.js — pure client-side state, no server round-trip needed. Use Alpine for modals, dropdowns, tabs, and other local UI state. Use htmx for anything that needs data from the server.

  • Toggle, modals, tabs → Alpine.js
  • Partial page updates from server data → htmx
  • Full page loads → FastAPI + Jinja2
